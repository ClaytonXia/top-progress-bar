### top progress bar plugin for jquery
---

#### Usage

1. import plugin
```html
    <script src="jquery.min.js"></script>
    <script src="top-progress-bar.min.js"></script>
```

2. init plugin
```javascript
<script>
$(document).ready(function() {
    $(document).topProgressBar({
        bgColor: "red",
        height: "3px"
    });
});
</script>
```
